# SEDAI Solar2Grid Mobile App

Flutter mobile application for solar energy management.

## Getting Started

1. Install Flutter SDK
2. Run `flutter pub get`
3. Run `flutter run`

## API Configuration

Update `lib/core/api/api_client.dart` with your API endpoint:
- Android emulator: `http://10.0.2.2:8000`
- iOS simulator: `http://localhost:8000`
- Real device: Use your computer's local IP
